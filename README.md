# 常见的中文术语翻译
要求：
- 全称： 字数不限，意译
- 简称：字数不多于其英文缩写音节数+1， 意译

## 深度学习
| 英文名称或者缩写        | 中文翻译 |
|----------------------|---------|
| CNN                  | 卷积网   |
| convolutional layers | 卷积层   |
| RNN                  | 递归网   |
| recurrent layers     | 递归层   |
| GNN                  | 图网络   |
| LSTM                 | 长记忆层 |
| Transformer          | 变换器   |
| attention            | 注意力   |
